orientdb-binary-protocol-implementation
=======================================

- This project is an implementation of [OrientDB's binary protocol](https://orientdb.com/docs/2.2.x/Network-Binary-Protocol.html)
in Java.

Getting Started
===============

```
mvn clean install
```
